DOCUMENTATION
Akatab (Tifinagh script)
========================

This file describes the documentation files included with the Akatab 
font. This information should be distributed along with the Akatab font 
and any derivative works.

The Akatab font is on the Akatab website (https://software.sil.org/akatab/).

The Akatab-Character-Inventory.odt file gives a sample of all the Unicode 
characters in the font (source file for the pdf).

The Akatab-Character-Inventory.pdf file gives a sample of all the Unicode 
characters in the font.

The Akatab-documentation.odt file illustrates the use of OpenType and 
Graphite features. It also documents the bi-consonant ligatures as well 
as right to left and variant glyphs (source file for the pdf).

The Akatab-documentation.pdf file illustrates the use of OpenType and 
Graphite features. It also documents the bi-consonant ligatures as well 
as right to left and variant glyphs.
